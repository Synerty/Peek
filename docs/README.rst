synerty-peek
------------

A platform for python written for Twisted.

This is the meta package that installs the platform.

.. NOTE:: If you don't want to install the whole platform, you can install just the
    service you want.

Services
--------

*  peek_server
*  peek_worker
*  peek_agent
*  peek_client
