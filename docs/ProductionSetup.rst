Windows
-------

Run the command prompt as administrator then enter the bash shell.

#.  synerty-peek::

        $ pip install synerty-peek

#.  Install front end packages::

        $ cd `dirname $(which python)`/lib/site-packages/peek_client_fe
        $ npm install
